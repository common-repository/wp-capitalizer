This plugin  is used for change the post title to capitalize automatically.

== Description ==
This is a plugin used to change the post title to capitalize automatically.if we want to change the post title to capitalize,just activate this plugin.it will automatically convert the post title to capital letter.

== Installation ==
1. Search \'WP-capitalizer\' to Plugins library.
2. Install the plugin by click the Install option.
3. Activate the plugin.
4. That\'s all. Its working automatically.
OR
1. Download the \'WP-capitalizer.zip\' file from WordPress plugins directory.
2. Upload to \'/wp-content/plugins/\' directory
3. Unzip the plugin folder
4. Activate the plugin through the \'Plugins\' menu in WordPress Dashboard.
5. That\'s all. Its working automatically.
